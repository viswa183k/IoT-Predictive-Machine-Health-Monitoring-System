# IoT Predictive Machine Health Monitoring System

This project implements an Industry 4.0 predictive maintenance system using ESP32, MPU6050, and DHT22 sensors. It continuously monitors machine vibration, temperature, and operating hours, uploads data to ThingSpeak, and alerts operators when abnormal conditions indicate possible equipment failure.

## Features

- Real-time vibration monitoring
- Machine temperature monitoring
- Operating hour tracking
- Cloud-based monitoring using ThingSpeak
- Automatic abnormal condition detection
- Predictive maintenance support

## Hardware Required

- ESP32 Development Board
- MPU6050 Accelerometer
- DHT22 Temperature Sensor
- LED or Buzzer
- WiFi Connection
- Breadboard & Jumper Wires

## Applications

- Smart factories
- Manufacturing industries
- CNC machine monitoring
- Industrial motors
- Predictive maintenance systems

## Future Improvements

- MQTT integration
